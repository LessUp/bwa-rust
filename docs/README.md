# Internal Documentation

`docs/` is intentionally small. Public user-facing material lives on GitHub Pages under `site/`; normative behavior lives under `openspec/specs/`.

## Current Tree

- `docs/development/ai-workflow.md`: project-specific AI/OpenSpec workflow.
- `docs/development/tooling.md`: local editor, CLI skill, and verification tooling.
- `docs/superpowers/plans/`: agent execution plans created during large changes.

Historical tutorials and architecture duplicates were removed during closeout. Use the Pages portal for user guidance and `openspec/specs/` for requirements.
