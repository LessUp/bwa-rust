# 更新日志

所有重要的项目变更都将记录在此文件中。

## [v0.3.0] - 🚧 开发中

### 新增

- **配对端比对支持**（开发中）
  - 配对 FASTQ 解析（分离文件和交错格式）
  - Insert size 估计（中位数和 MAD）
  - 自动阈值更新
  - PE FASTQ 验证和错误检查

### 规范文档

- PE alignment 产品规范（F-011 至 F-017）
- RFC-0004：配对端比对技术设计
- 详细实施计划（5 个阶段）

## [v0.2.0] - 2026-04-17

### 文档

- **双语文档套件**
  - 完整的文档重构，支持中文和英文
  - 架构文档（概述、索引构建、比对算法）
  - 教程文档（快速入门、算法教程）
  - API 文档（库使用指南）

### 新增

- **内存保护配置**
  - `max_occ`（默认：500）- 跳过出现次数过多的种子
  - `max_chains_per_contig`（默认：5）- 限制每个 contig 的链数
  - `max_alignments_per_read`（默认：5）- 限制每个 read 的比对数
  - `AlignOpt::validate()` 参数验证方法

### 修复

- **比对质量**
  - 修复强反向命中的候选过滤
  - 添加半全局精细化，提高 CIGAR/NM 准确性
  - 引入 clip 惩罚用于候选排序

- **输入验证**
  - FASTA 头部验证
  - 空序列检测
  - 重复 contig 名称拒绝
  - 线程数验证

### 变更

- **代码质量**
  - 提取命名常量，消除魔法数字
  - 为公共 API 添加全面的文档注释
  - 优化热路径分配

## [v0.1.0] - 2026-02-13

### 新增

- **索引构建**（`index` 子命令）
  - FASTA 解析器，支持多 contig
  - 后缀数组构建（倍增算法）
  - BWT 构建
  - FM 索引与稀疏 SA 采样
  - 单一 `.fm` 索引文件

- **序列比对**（`align` 子命令）
  - SMEM 种子查找
  - 种子链构建与 DP 评分
  - 带状 Smith-Waterman 比对
  - 通过 rayon 实现多线程
  - 完整 header 的 SAM 输出

- **一步比对**（`mem` 子命令）
  - BWA-MEM 风格工作流
  - 默认 BWA-MEM 评分参数

- **工程化**
  - 175 项测试（单元 + 集成）
  - GitHub Actions CI/CD
  - Criterion 基准测试
  - 文档站点
