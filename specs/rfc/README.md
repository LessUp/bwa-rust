This directory contains technical design documents and architecture proposals.

|-----|-------|--------|-------------|
| [0001](./0001-core-architecture.md) | Core Architecture | ✅ Accepted | Module architecture, data flow, tech stack |

## RFC Naming Convention

RFC documents follow the naming pattern: `NNNN-title.md`

- `NNNN`: Four-digit serial number (e.g., 0001, 0002)
- `title`: Short kebab-case title

## RFC Status

| Status | Meaning |
|--------|---------|
| 📝 Draft | Work in progress |
| 👀 Review | Under review |
| ✅ Accepted | Approved and implemented |
| ❌ Rejected | Not approved |
| 🗑️ Withdrawn | Withdrawn by author |
